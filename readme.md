# LatinNow Project data

## Description

This repo consists of the database of inscribed epigraphical objects created by the LatinNow project (https://latinnow.eu) and used as the source material for the open access volumes published by the LatinNow team [Link][1] during the course of an ERC-funded project running from 2017-2023.


## Getting Started

Please note that at present, this repo consists only of the database table exports. The DDL of the database schema, a data dictionary, and further information about how to indepdently recreate the database are still in develpoment and should be added to this respository in the near future.
